# tugas-4-kriptografi
Implementasi Algoritma RSA, ElGamal, Paillier, ECC

# Prerequisite
- Python 3.9
- PySimpleGUI, installation via pip `pip install PySimpleGUI`

# How To Run
- Open terminal in root directory and `python gui.py` 
- Double click `gui.py`

# Notes:
- Keys will be saved in `keys` folder
